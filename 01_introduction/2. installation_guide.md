# Python Installation Guide (SQA-Oriented)

This guide explains how to install and configure Python for Software Quality Assurance (SQA) tasks such as automation testing, scripting, and test framework setup.

---

## 1. Purpose of Python for SQA

Python is widely used in SQA for:
- Automation testing (e.g., Selenium, Playwright)
- API testing (e.g., requests, pytest)
- Performance testing scripting
- Test data generation and processing

---

## 2. System Requirements

- Operating System: Windows / macOS / Linux
- Minimum: 4 GB RAM recommended
- Internet connection (for installing packages)

---

## 3. Download Python

Download Python from the official website:

https://www.python.org/downloads/

- Choose the latest stable Python 3 version
- Avoid outdated versions (Python 2.x is deprecated)

---

## 4. Install Python

### Windows Installation

1. Run the downloaded `.exe` file
2. IMPORTANT: Enable the checkbox:
   "Add Python to PATH"
3. Click "Install Now"

Optional (Advanced Setup):
- Customize installation path if needed
- Ensure pip is selected

---

### macOS Installation

1. Open the `.pkg` installer
2. Follow the installation steps
3. Python will be installed automatically

---

### Linux Installation

Most Linux systems already include Python.

# 5. Check Version
python3 --version

# 6. Verify Installation

### Open terminal or command prompt and run:
python --version
python3 --version
